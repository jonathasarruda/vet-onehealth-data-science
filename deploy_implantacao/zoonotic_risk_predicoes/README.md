# Zoonotic Risk Predictions App

This repository contains a Streamlit application to visualize zoonotic risk predictions by farm. The app was built with Streamlit, pandas, and
Plotly, and is part of the vet-onehealth-data-science project. It generates interactive charts showing predicted risk levels across farms,
supporting decision-making in veterinary and public health. To run locally, clone the repository, 
navigate to deploy_implantacao/zoonotic_risk_predicoes, install dependencies 
with pip install -r requirements.txt, and run streamlit run app_predicoes_risk_outbreak.py. The app is also publicly available 
via Streamlit Cloud at https://zoonotic-risk-predicoes-app.streamlit.app/
