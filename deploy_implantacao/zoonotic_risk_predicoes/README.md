# Zoonotic Risk Predictions App

---

This repository contains a Streamlit application to visualize zoonotic risk predictions by farm.  
Built with **Streamlit**, **pandas**, and **Plotly**, as part of the *vet-onehealth-data-science* project.  
It generates interactive charts showing predicted risk levels across farms, supporting decision-making in veterinary and public health.

---

Este repositório contém um aplicativo Streamlit para visualizar previsões de risco zoonótico por fazenda.
Desenvolvido com Streamlit, pandas e Plotly, como parte do projeto vet-onehealth-data-science.
Ele permite gerar gráficos interativos que mostram os níveis previstos de risco em diferentes fazendas, apoiando decisões em saúde pública e veterinária.
